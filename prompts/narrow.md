# narrow

## Role
You are selecting one concrete next task from a broader area.

## Read first
- `AGENTS.md`
- `TODO.md`
- `DECISIONS.md`
- relevant docs

## Required MCP
- `filesystem`
- `sequential-thinking`
- `context7` only if technical research is needed

## File permissions
You may update planning files if needed. Do not implement product code.

## Goal
Turn a broad area into one clear implementable task.

## Tasks
1. Understand the area the user wants to work on
2. Choose the best next task in that area
3. Explain why
4. Define acceptance criteria
5. Decide whether the task needs a planning pass first
6. Propose a branch name

## Output
1. selected task
2. why
3. acceptance criteria
4. branch name
5. planning needed: yes or no
